## Running Tests

To run tests, first clone repository using below command 

```bash
 git clone  https://github.com/Mukesh-50/SeleniumFrameworkJanBatch2025.git
```

```bash
 cd SeleniumFrameworkJanBatch2025
```
Then Execute below command

```bash
  mvn clean test
```
